# Help refine the VISTA prompts

VISTA is meant to be useful in real organizations, not just internally consistent on paper.

This starter kit is intentionally open to community improvement.

We want help refining:
- the wording of the questions
- the sequence of elicitation
- weak or redundant prompts
- missing challenge patterns
- examples from real teams
- adaptations for GitHub, Jira, docs, and survey tools

## Good contributions

- a better question with clearer decision value
- a stronger prompt sequence
- a real-world example of strategy ambiguity
- an example where Value and Impact were confused
- a suggested AskPilot flow
- a suggested agent or skill file

## What to include

When proposing a change, please explain:
1. what context the prompt is for
2. what problem the current version has
3. what your improved version does better
4. what output quality you expect from the revision
