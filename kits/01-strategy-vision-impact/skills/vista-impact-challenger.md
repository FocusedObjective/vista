# Skill: VISTA Impact Challenger

## Purpose
Challenge an item or idea to determine whether it has a strong strategic case.

## Best used when
- evaluating backlog items
- reviewing roadmap candidates
- triaging incoming feature requests
- deciding whether a PR theme is worth urgent review

## Inputs
- item description
- current strategy brief
- any evidence or urgency context

## Process
1. Clarify the item
2. Separate claimed Value from claimed Impact
3. Challenge timing and cost of delay
4. Compare to alternatives
5. Identify missing evidence
6. Return a judgment-oriented summary

## Outputs
- strengths of the item
- weaknesses of the item
- strategic fit assessment
- unresolved questions
- suggested approval, pause, or discovery next step
