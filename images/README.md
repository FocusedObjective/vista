## Image assets

This folder contains SVG teaching diagrams for the VISTA framework.

- `vista_dimensions.svg` - overview of the framework, decision rule, operating layers, and metric timing
- `vista_odim.svg` - ODIM and MIDO diagram for designing useful metrics
- `vista_principles.svg` - grouped view of the ten principles
- `vista_metrics_matrix.svg` - matrix of example metrics by dimension and timing category
- `vista_dashboards.svg` - dashboard cuts for executives, delivery leaders, and teams

These images are designed to work well in GitHub previews and documentation embeds.
