# Work Should Be Judged Comparatively, Not in Isolation

## Principle

**Work should be judged comparatively, not in isolation.**

The question is not only whether a work item is good. The more important question is whether it is better, right now, than the alternatives competing for the same scarce capacity.

## Why this matters

A work item can be good and still be the wrong choice if a better option exists.

## Summary

VISTA treats prioritization as comparative choice under constraint.
